The Circuit Footprint

Definition

A curated pre-runtime feature vector that transforms a circuit graph into a 1D array — interpretable, auditable, and reusable across ML models.

Classical Analogy
Cyclomatic complexity → defects
Halstead metrics → effort
Circuit Footprint → execution quality

Three feature configurations evaluated to assess hardware context (H3):

Circuit-only

Pre-runtime metrics from static analysis (after transpilation).

Hardware-only

Backend calibration features (gate errors, $T_1$ / $T_2$).

Mixed

Circuit footprint + calibration features.
