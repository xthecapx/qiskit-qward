Algorithm Selection

Criterion: algorithms with expected output states known a priori — so fidelity metrics can be computed against ground truth.

Fourier-based

QFT — Quantum Fourier Transform

PE — Phase Estimation

Exploits periodicity in quantum amplitudes to extract phase information.

Oracle-based

GO — Grover's Operator (search)

BV — Bernstein-Vazirani

Black-box oracle + interference reveals hidden information encoded in a function.

Entanglement-based

vTP — variation of Teleportation Protocol

GHZ — GHZ state preparation

Creates and distributes entanglement — baseline for studying entanglement degradation at scale.

Why these six?
  • Known analytically
  • Superposition
  • Entanglement
  • Interference
  • Measurement
Excluded
  • VQE (Variational Quantum Eigensolver)
  • Hamiltonian simulation

Outputs lack a discrete set of dominant states — would require full-distribution analysis, out of scope.
